## Claude가 지킬 것
- **검증은 시뮬레이터 기본**: 반복 확인은 `xcrun simctl install/launch/io screenshot`(내가 직접 스크린샷 보고 검증). 실기기는 최종 + 디바이스 전용(마이크/STT/성능)에만.
- **외부 의존 2회 실패 = 즉시 전환**: 잠긴 폰·플래키 터널에 매달리지 말고 통제 가능한 수단으로.
- **'너한테 필요한 것'은 앞에서 한 번에**: 결정·크레덴셜·승인을 작업 중간에 찔끔찔끔 묻지 말고, 시작에 몰아서 1회.
- **빡친 신호엔 짧게, 행동 먼저**: 긴 글 금지. 1~3줄 + 바로 실행.
- **한 덩어리 → 커밋+로그 1회 → 다음**: 중간 갈아타기 최소화, 로그 훅 사이클 최소화.
